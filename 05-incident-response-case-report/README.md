## Overview
This project demonstrates a SOC-style incident response workflow using
endpoint telemetry collected via Sysmon on a Windows 11 system.

## Objectives
- Reconstruct an incident timeline using event logs
- Identify indicators of compromise (IOCs)
- Document analysis and response actions
- Apply MITRE ATT&CK mappings

## Tools Used
- Sysmon
- Windows Event Viewer
- PowerShell

## Outcome
The incident was analyzed, documented, and closed as benign lab activity,
demonstrating proper SOC triage and response procedures.
