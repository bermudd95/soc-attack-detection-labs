# Threat Analysis & Vulnerability Assessment (CISA-Aligned)

## Overview

This lab focuses on threat intelligence analysis using publicly disclosed
vulnerabilities and advisories from the Cybersecurity and Infrastructure
Security Agency (CISA).

The objective is to assess real-world vulnerabilities, evaluate business risk,
and provide actionable remediation guidance.

---

## Objectives

- Review active CISA advisories
- Identify affected systems and threat actors
- Assess likelihood and impact
- Propose mitigation and remediation strategies

---

## Methodology

1. Reviewed selected CISA advisories
2. Identified vulnerability characteristics and exploitation methods
3. Mapped threats to MITRE ATT&CK techniques
4. Assessed organizational impact
5. Drafted remediation guidance
